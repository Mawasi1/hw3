# Express/MongoDB API Backend.
### installtaion
***
- How to run the code in your local machine:
```
$ git clone https://github.com/Mawasi1/hw2-express-mongo.git
$ cd hw2-express-mongo
$ npm run server
```
- testing with jest: 
```
$ npm test
```
